apt

apt(Advanced Packaging Tool)是一个包管理器,用于在Ubuntu,Debian和相关Linux发行版上安装,更新,删除,管理软件包

centos系统默认包管理器为yum

通过sudo apt-get update && sudo apt-get upgrade,可以更新升级软件等


区别在那¿

update:

在不改变现有软件设置的基础上更新软件列表,同步 /etc/apt/sources.list 和 /etc/apt/sources.list.d 中列出的源的索引




upgrade:

把本地已安装的软件，与刚下载的软件列表里对应软件进行对比,如果发现已安装的软件版本太低,就会更新

试试这个:sudo apt-get update &&sudo apt-get upgrade
